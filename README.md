# web-services

This is an example on how to build a modern reactive system which comunicates with a "legacy" system (SOAP Protocol). In order to learn new things about SOAP Protocol in Spring and Reactive WebFlux I created a simple application which adds and employee to a MySQL database (WS SOAP). This system is called from a Reactive System (Special thanks to [gungor] who implemented a soap reactive client)
